Sistema de Biblioteca
Descrição
Este projeto consiste em um MVP (Produto Mínimo Viável) de um Sistema de Biblioteca desenvolvido para a disciplina de Analise desenvolvimento de sistemas.

O sistema permite gerenciar livros, usuários e empréstimos, além de oferecer consulta de livros e um formulário de contato.

Tecnologias utilizadas
Node.js
Express
HTML5
CSS3
JavaScript
Funcionalidades
Login
Dashboard
Cadastro de Livros (CRUD)
Cadastro de Usuários (CRUD)
Empréstimos e devoluções
Consulta de livros
Formulário de contato
Como executar
Instale as dependências:
npm install

Execute o servidor:
npm run dev

Abra no navegador:
http://localhost:3000

Autor
Projeto acadêmico desenvolvido para a disciplina de Engenharia de Software.
